Development Version from Git

The official Git repository is hosted by Github at 

  https://github.com/gccxml/gccxml
One may clone it with the command 
  git clone git://github.com/gccxml/gccxml.git
Git may be obtained from the Git Home Page.

Other Files

  • A stylesheet contributed by Ivan Kissiov to help display the dumped xml: gccxml.xsl.
  • An unofficial DTD contributed by Paul Ross and Martin Piskernig: gccxml-2004-11-19.dtd.txt. No attempt will be made to keep the output conforming to this DTD and it has not been checked by a GCC-XML author.
